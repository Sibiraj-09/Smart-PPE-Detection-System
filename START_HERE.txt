📊 SYSTEM FLOW

Input Video
    ↓
[YOLO Detection]
  - Detect people
  - Detect helmets
  - Detect vests
    ↓
[Zone Detection]
  - Person crosses line1?
  - Lock person
  - Ignore others
    ↓
[Wait RFID]
  - Listen to ESP32
  - Read RFID UID
  - Timeout: 5 seconds
    ↓
[Decision]
  - helmet == YES && vest == YES?
  - Status: SAFE or UNSAFE
    ↓
[Hardware Signal]
  - Send G (green) if SAFE
  - Send R (red+buzz) if UNSAFE
  - Wait 2 seconds
  - Send O (off)
    ↓
[Save Data]
  - worker_id (RFID UID)
  - helmet (YES/NO)
  - vest (YES/NO)
  - status (SAFE/UNSAFE)
  - gate (from filename)
  - time & date
    ↓
Output Video + Database Records

📞 TROUBLESHOOTING

RFID not detected:
  • Check Device Manager for COM port
  • Close Arduino Serial Monitor
  • Verify ESP32 firmware uploaded
  • Run: python test.py

Port 5000 already in use:
  • taskkill /IM python.exe /F
  • Or change port in app.py

Model not found:
  • Verify best.pt path in detection.py
  • Run: python test.py

Video upload fails:
  • Check file format (MP4, AVI, MOV)
  • Reduce file size
  • Verify backend/uploads folder exists

Processing very slow:
  • Use smaller videos
  • Reduce video resolution before upload
  • Or wait for it to complete
