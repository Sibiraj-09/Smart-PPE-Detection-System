ESP32 HARDWARE FIRMWARE:
════════════════════════════════════════════════════════════════════════════════

File: esp32/esp32_rfid_led_buzzer.ino
Size: ~6 KB (140 lines)
Purpose: ESP32 microcontroller firmware
Features:
  ✓ MFRC522 RFID reader initialization
  ✓ Serial communication at 115200 baud
  ✓ RFID card scanning and UID reading
  ✓ LED control on pins D12 (Red) and D14 (Green)
  ✓ Buzzer control on pin D4
  ✓ Command parsing from Python (G/R/O)
  ✓ Status LED pulsing for buzzer
  ✓ Pin configuration comments

Led Control Commands:
  'G' → Green LED ON (Green anode D14 to 3.3V)
  'R' → Red LED + Buzzer ON (Red anode D12 to 3.3V, Buzzer D4 pulse)
  'O' → All OFF

Pin Assignments:
  RFID Module:
    SDA → D5
    SCK → D18
    MOSI → D23
    MISO → D19
    RST → D22
  LEDs:
    Red LED → D12 (with 220Ω resistor to GND)
    Green LED → D14 (with 220Ω resistor to GND)
  Buzzer:
    Positive → D4 (with resistor if needed)
    Negative → GND
  Power:
    VCC (RFID) → 3.3V
    GND → Common ground (all components)
